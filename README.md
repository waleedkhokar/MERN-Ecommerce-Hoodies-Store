## 🚀 About This Project

This is the **frontend** of an eCommerce web app for selling premium hoodies.  
It focuses on **clean UI/UX**, performance, and modern design with dark/light mode.

⚠️ This public repo contains **Frontend only**.

The **full version** includes:
- 🔐 Admin Panel (Products, Orders, Users, Reviews)
- 🛒 Cart & Checkout
- 📦 Order Management
- 👤 User Authentication
- 🖼 Image Upload System
